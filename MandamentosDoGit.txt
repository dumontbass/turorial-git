git init - inicia um repositório.
git add - adiciona arquivos no repositorio.
git status - mostra o status do repositório.
git commit -am -  Checa suas alterações no repositório.
git push origin master - enviar código no repositório.
git pull origin master - baixar alterações no repositório.
